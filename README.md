# CBPQ

An App for skydivers in Brazil to check their license status.

## Getting started

1. Create `.env` 

2. Set JKS path property `storeFile` in `/android/key.properties`

3. Install the dependencies:

```
flutter packages get
```

4. Run the app:

```
flutter run
```
